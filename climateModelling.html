Climate Modelling

Climate modelling is the result of essential computations that benefits society greatly.
It allows for weather forecasts and informed government planning.

It starts by collecting data inputs from various sources, such as satellites and weather stations.
Raw data gathered is consolidated for processing.

During processing, data is cleaned - removing errors and amending any missing data if necessary.
Data may also need to be changed - data formats and the size of the data itself, so that it can be processed.

High performance computers run complex mathematical models using gathered data to produce simulations and predictions.

Outputs of large data sets can be analysed using statistical analysis tools to identify trends and possible hazards.

Processed data is used to generate visualisations of the data - such as graphs, animations and weather maps.
This can then be used by meteorologists to interpret data easily and produce reports.

Through this process, weather trends can easily be identified, to inform society on the weather forecast,
and unknown hazards can be discovered, to give governments the necessary information to act accordingly and potentially save lives.
